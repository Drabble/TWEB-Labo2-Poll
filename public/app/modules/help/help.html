Navigation

You can see the top navigation bar in the image below. On the left you can toggle on and off the left sidebar.

The Login and Register features can be accessed using the corresponding buttons on the top right corner. The Settings icon allows the user to customize the layout of the application as you can see in the image below


The left sidebar in the image below allows you to join an existing room by specifying its id. You can also access the homepage by clicking on the Home button.

Accounts

The Login and Register features can be accessed using the corresponding buttons on the top left corner of the page.

To View your profile information or to Log out of the app, the user can click on its Username in the top right corner. This displays the Sign out button and a Profile button to view your account informations.


Rooms

To Create a new room you must first be logged in, then you can click on the Create room button on the left sidebar.

To Join an existing room, the user can fill the Join a room text field, in the top left corner, with the id of the room and click on the appended button to join it or press enter.

The left part displays, for logged in users, the Rooms that the user has previously created
