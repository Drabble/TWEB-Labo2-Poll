Welcome to Pollspeak

Pollspeak is a webapp allowing you to create and participate in interactive polls

Login/Register and create new rooms

You can join create an account, login and create new rooms.

Join rooms and participate

You can join existing rooms and participate.

Ask questions

You can create new questions inside rooms.

Answer questions

You can comment the questions inside rooms.

Like/Dislike questions

You can like and dislike the questions.

Pollspeak is free and open source

Pollspeak is opensource. You can find Pollspeak on Github.

Pollspeak is responsive

You can view it on any browser and any device.

Pollspeak is customisable

You can customise your Pollspeak layout and colors.

About us

Our developement team is composed of Antoine Drabble and Guillaume Serneels. Antoine Drabble is the main developer and Guillaume Serneels is the software quality assurance engineer. We created this project for the TWEB course at HEIG-VD and our goal is to deliver a professional polling web app.
