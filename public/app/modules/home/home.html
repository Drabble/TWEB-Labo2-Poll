Interactive polls web application under developement

Welcome to the preview of the HEIG TWEB Polling web app.
The goal of this web app is to allow its users to participate in interactives polls
The visual interface is functional so you can already see how will look the different features which will be implemeted.

Landing page and app navigation

The landing page which you are currently viewing contains the different controls to navigate between the applications features, and it displays informations about the application's usage.

The Login and Register features can be accessed using the corresponding buttons on the top left corner of the page. The Settings icon allows the user to customize the layout of the application

To Create a new room you must first be logged in, then you can click on the Create room button on the left sidebar.

To Join an existing room, the user can fill the Join a room text field, in the top left corner, with the name of the room and click on the appended button to join it. If no room name has been provided, a click on this button redirects the user to the Room creation section.

The left part displays, for logged in users, the Rooms that the user has previously joined

To View and Edit its profile information or to Log out of the app, the user can click on its Username in the top right corner. This displays the Sign out button and a Profile button to view and edit its account informations.


Features

Feature 1 : Register and Login

Both of these page contain a form allowing the user, respectively, to fill its personnal information to create a new account, and to submit its credentials to log into the app.

Feature 2 : Room creation

In the form contained in this page, the user can fill in the room's name and choose if the room is persistent or temporary using the Temporary check-box. Feature 3 : Poll creation

When viewing a Room, users are able to ask new question by submitting the Ask question form

Feature 4 : Poll usage

Inside a Room, each question is diplayed and users can comment, upvote or downvote each question using the appropriate buttons.

Feature 5 : Account management

As a logged in user, a click on the Username in the top right corner, displays two option: a Sign Out button, to log out of the application, and a Profile button which displays the detailed informations about the user's account.
